A01 Binding

바인딩 식 내부에서는 식만 사용 가능 /

1. 일반적 바인딩
일반적인 단방향 바인딩: {{name}}
함수의 리턴값: {{onAdd(10,5)}}
함수의 리턴값: {{onMin(10,5)}}
배열: {{arr[0]}} / {{arr[1]}} / {{arr[2]}}
객체: {{user.name}} / {{user.age}} / {{user.address}}

2. 바인딩 연산
일반적 연산: {{10 + 20}}
속성 참조: {{ arr.length}}
속성 참조 연산: {{ arr.length*100 }}
비교 연산: {{name == 'Nolbu'}}
비교 연산: {{arr[0] > 10 && arr[1] > 10}}

3. 바인딩 관련 지시자
바인딩: {{text}}
v-text: 이 값은 무시됨(지워짐)
v-html: 지시자의 값은 항상 변수다
v-once: {{text}}
v-pre: {{text}}


CLI에서는 에러 Data에 존재하지 않는 변수: {{age}}
Data에 존재하지 않는 객체: {{obj && obj.name}}
