Vue 설치
    npm info @vue/cli   => 확인
    npm i -g @vue/cli   => 설치 (업그레이드)

설치 버전 확인
    vue --version

프로젝트 생성 - 이름은 소문자만 가능 특수문자 _, $ 가능
    vue create [productName]

프로젝트 시작
    cd productName
    npm run serve -> package.json 파일 참조

프로젝트 빌드
    npm run build -> package.json 파일 참조

build 후 확인
    npm i -g serve    => 모듈 설치

    serve -s dist     => 확인
